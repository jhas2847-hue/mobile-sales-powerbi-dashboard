# 📊 Mobile Sales Analysis Dashboard – Power BI

## 📌 Project Overview
This project presents an interactive **Mobile Sales Analysis Dashboard** built using **Power BI**. The dashboard helps analyze mobile phone sales performance across different brands, regions, and time periods.

The objective of this project is to convert raw sales data into meaningful insights using data visualization and DAX calculations. The dashboard allows users to explore key business metrics and understand sales trends in an intuitive way.

---

## 🎯 Project Objectives
• Analyze overall mobile sales performance  
• Track Month-to-Date (MTD) sales trends  
• Identify top-performing mobile brands  
• Compare sales across different regions  
• Visualize sales patterns over time  

---

## 📊 Dashboard Features
The Power BI dashboard includes the following components:

**1. Sales Overview**
- Total sales performance
- Overall revenue insights

**2. Month-to-Date (MTD) Analysis**
- Current month sales performance
- Comparison of sales trends

**3. Brand-wise Sales**
- Performance comparison of different mobile brands

**4. Regional Sales Analysis**
- Sales distribution across different regions

**5. Time-based Sales Trends**
- Monthly or periodic sales trend analysis

---

## 🛠 Tools & Technologies Used
• **Power BI** – Dashboard creation and visualization  
• **DAX (Data Analysis Expressions)** – Calculated measures and KPIs  
• **Data Cleaning** – Preparing raw data for analysis  
• **Data Visualization** – Creating interactive charts and insights  

---

## 📈 Key Insights
